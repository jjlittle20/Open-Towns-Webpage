A community project for fans of Towns

Towns had a unique mix of settlement building, dungeon exploration, resource management, and emergent chaos. Open Towns exists to preserve that experience, make the game easier to run on modern systems, and gradually improve the parts that were left unfinished or unstable.

Project Goals

Preserve, fix, document, improve

Bug Fixes

Community contributors are working through crashes, broken behaviours, long-standing issues, and rough edges in the original game.

Modern Compatibility

The project aims to make Towns easier to build, run, and maintain on modern systems and development tools.

Source Preservation

With the original source code released, fans can now study, preserve, and improve the game instead of letting it disappear.

Careful Improvements

Open Towns is focused on respectful improvements that keep the spirit of the original game intact.

Documentation

Part of the work is making the codebase easier to understand so more people can contribute fixes, tests, tools, and improvements.

Open Collaboration

Developers, players, modders, testers, and long-time fans can help shape the future of the project.
